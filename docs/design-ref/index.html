Cognest

AI 认知工作台 — 帮助你将碎片化灵感逐渐演化为系统化认知。人在两端，AI 在中间。

发现
AI 主动推送发现：本周回顾、成长中的 Topic、新关联。支持按需生成视图（知识图谱、趋势、阅读清单）。
创作
文件列表 + Markdown 编辑器 + AI 对话面板。写作时自动浮出相关碎片，支持沉浸模式。
碎片
Inbox 时间流，按日分组。零摩擦录入，AI 自动分类和关联。支持全局快捷键 ⌘⇧Space。
文章
全部文章管理：分类标签、状态追踪（草稿/修改中/已完成）、搜索筛选、预览、跳转创作。
